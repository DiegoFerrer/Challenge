# Challenge-Reign
## Download repository

```bash
# Clone repository
$ git clone https://github.com/DiegoFerrer/challenge.git

# Open terminal in path repository
  yourPath/yourPath/challenge
```

## Running the app

```bash
# build images docker
  docker-compose build

# up containers
  docker-compose up
```

## View the app

```bash
# Open your navigator (chrome,firefox,etc)
  http://localhost:4200

# Only User y password accepted:
  user: admin
  password: admin
```
